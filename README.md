Jinxes
======

Simple python curses appliction library

You will need a 256 color terminal with a unicode font.

Sample Apps
===========

Press q or ctrl-c to exit the app.

Flying Toasters
---------------

    ./flying-toasters

A curses screensaver inspired by the AfterDark screensaver of the same name.

Tag The Flag
------------

    ./tag-the-flag

Reach the goal before the monsters get you.  Be careful, they bounce now!

Move with hjkl
